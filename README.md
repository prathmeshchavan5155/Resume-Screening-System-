# AI-Based Resume Screening System

## Description

An intelligent resume screening system using NLP and Machine Learning to automatically rank job candidates based on skill match and job relevance.

## Features

* Resume Text Processing
* NLP Preprocessing
* TF-IDF Vectorization
* Cosine Similarity Matching
* Resume Ranking
* Match Percentage Calculation
* Streamlit Web Interface

## Tech Stack

* Python
* Pandas
* NLTK
* Scikit-learn
* Streamlit
* Matplotlib

## Workflow

1. Load Resume Dataset
2. Text Preprocessing
3. TF-IDF Feature Extraction
4. Job Description Input
5. Cosine Similarity Calculation
6. Resume Ranking
7. Display Results in Streamlit

## Installation

pip install -r requirements.txt

## Run Project

streamlit run app.py

## Model Used

* TF-IDF Vectorizer
* Cosine Similarity

## Output

* Ranked Resumes
* Similarity Score
* Match Percentage
